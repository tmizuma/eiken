# 英検一級 Reading Practice

英検一級レベルの単語学習と長文読解練習ができる Web アプリです。

## セットアップ

```bash
# 依存関係のインストール
cd app && npm install

# DB の初期化 (初回のみ)
make db-reset
```

## 起動

```bash
make dev
```

http://localhost:3000 でアクセスできます。

## 機能

### 単語一覧 (/words)

- 1939 語の英検一級単語を一覧表示
- 単語・意味で検索可能
- 「覚えた単語を非表示」フィルタで未習得の単語だけに絞れる
- 一覧から直接「覚えた」をマークできる

### 単語詳細 (/words/[id])

- 発音記号、カタカナ読み、例文 2 つを表示
- 類義語・反語へのリンク
- 「覚えた!」ボタンで習得状態をトグル

### 長文問題一覧 (/passages)

- トピック別にタグ表示 (政治/科学/心理/文化 等)
- クリックでクイズページへ

### 長文クイズ (/passages/[id])

- 左に長文、右にクイズ (4 択 x 3 問) の分割レイアウト
- クイズを閉じると長文が全幅に広がる
- 長文中の英検一級単語はリンクで詳細ページへ飛べる
- 全問回答後「回答する」で解説ページへ

### 解説ページ (/passages/[id]/result)

- 正答数、英文、日本語訳、各問題の解説を表示
- 使用された英検一級単語の一覧

## 長文問題の追加

Claude Code のスキルで新しい問題を生成できます:

```
/eiken-reading トピック / 単語範囲
```

例: `/eiken-reading 科学 / 1-600`
