# ローカル英単語復習リストアプリ: AI エージェント向け仕様書

## 目的

ローカル完結で、英単語の学習ログを登録し、今日の日付から逆算した復習対象(昨日 / 今日-3 / 今日-7 / 今日-14)をトップページにテーブル表示する最小限の Web アプリを作る。

- 学んだ単語を全部総復習するのは避けたい
- その代わり「所定の間隔で復習すべき範囲」だけを表示したい
- クラウドホスト不要、ローカルのみ
- シンプル優先: フレームワーク不要(React/Next/Express 等は使わない)

## 実行環境/起動方法

- Node.js でローカルサーバを起動
- `npm run dev` で起動
- 同じディレクトリに SQLite の DB ファイルを配置(例: `app.db`)

## 技術制約

- UI: 素の HTML/CSS(最低限で OK)
- サーバ: Node.js 標準 `http` モジュールで実装
- ルーティング: 手書き(パスで分岐)
- DB: SQLite(ファイルベース)
- ORM 不要
- テンプレエンジン不要(文字列連結で HTML 生成で OK)
- ビルド不要
- ローカルブラウザからアクセスできれば OK

## 機能要件

### 1) 学習ログ登録

ユーザーが「今日学んだ範囲」を登録できること。

- 入力例: テキストエリア(markdown で記入)
- 学習日はデフォルトで「今日」(ユーザーが任意で変更もできると良いが、最小版は今日固定で OK)
- 登録したらトップページに戻る

### 2) トップページに復習リスト表示

トップページに、本日の日付から逆算して復習対象をテーブル表示する。

復習対象の列:

- `昨日` = 今日-1
- `今日-3` = 今日-3
- `今日-7` = 今日-7
- `今日-14` = 今日-14

表示仕様:

- 各列には「その日付に学習したログ」を一覧表示(0 件なら空欄)
- 日付が存在しない場合(例: 学習開始から 14 日未満)は空欄で OK
- テーブルには、少なくとも以下を表示:
  - textarea の初めの 20 文字
  - 学習日

### 3) 学習ログ一覧

最小要件では必須ではないが、可能なら `/logs` で全ログ一覧を見られるようにする。

## 画面/ルート仕様

### GET `/`

- トップページ
- 復習テーブル表示
- 「新規登録」リンクを表示

### GET `/new`

- 新規登録フォーム(HTML form)
- 入力: title, start_no, end_no
- submit 先: POST `/create`

### POST `/create`

- フォーム値を SQLite に INSERT
- 302 リダイレクトで `/` に戻す

### GET `/logs`

- 全ログの一覧表示

## データモデル(SQLite)

### DB ファイル

- 例: `app.db` (プロジェクト直下)

### テーブル: `study_logs`

### GET `logs/id`

詳細ページでマークダウンで書いた記入内容を閲覧、編集できること。つまり、markdown のパーサーが必要(最小限で良い)

カラム:

- `id` INTEGER PRIMARY KEY AUTOINCREMENT
- `study_date` TEXT NOT NULL (YYYY-MM-DD)
- `text` TEXT NOT NULL
- `created_at` TEXT NOT NULL (ISO 文字列で OK)

インデックス

- `CREATE INDEX idx_study_logs_date ON study_logs(study_date);`

## 日付ロジック

- サーバ側で「今日」の日付を生成
- タイムゾーンはローカルで OK(できれば `Asia/Tokyo` 想定)
- 文字列比較がしやすいよう `YYYY-MM-DD` で扱う
- 復習対象日:
  - d-1, d-3, d-7, d-14
- クエリ:
  - `SELECT * FROM study_logs WHERE study_date = ? ORDER BY id DESC;`

- 見た目は最小限で OK

## エラーハンドリング

- start_no <= end_no のバリデーション
- 未入力はエラーメッセージ表示
- DB 未初期化なら起動時にテーブル作成する

## 完成条件(Definition of Done)

- `npm install` → `npm run dev` で起動できる
- ブラウザで `http://localhost:3000/` が開く
- `/new` から学習ログを登録できる
- `/` に復習対象(昨日/3/7/14 日前)がテーブルで表示される
- DB はローカルの SQLite ファイルに保存され、再起動しても残る
